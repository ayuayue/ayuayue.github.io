记录log中的值到一个临时变量中

通常我们在chrome的devtools的工具中使用console.log来打印一些值,有时候是一个对象.但是我们想对这个值进行一个后续的操作,比如json转字符串,或者字符串的筛选操作,我们可以使用下面的操作自动创建一个临时变量进行保存

选中log输出的值,鼠标右键,保存为全局变量.自动会赋值给一个tmp的变量,序号是变量的个数.然后我们就可以使用tmp变量来代替log中的值来进行一些操作了.

使用快捷键

ctrl + L : 可以对控制台进行清屏

鼠标聚焦到浏览器中,使用 ctrl + L可以聚焦到url栏

alt + enter : 新窗口打开当前标签页,前提是已经聚焦到了url上,可以配合 ctrl +L alt + enter使用

ctrl + shift + c : 呼出选择dom元素的工具

ctrl + shift + m : 切换设备类型

esc : 在控制台中,使用 esc可以再打开一个控制台,再次使用则关闭
